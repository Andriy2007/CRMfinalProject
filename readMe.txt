Ласкаво просимо до серверної частини Школи програмування CRM! Цей проект служить базовим компонентом системи CRM,
надаючи надійний і ефективний API для керування заявками та запитами студентів. Завдяки Nest.js, TypeScript і інтегрованому з базою даних MySQL
це серверне рішення забезпечує безперебійне зберігання та пошук даних.

Технічна специфікація
Цей проект спрямований на розробку системи управління запитами для Школи програмування. Він використовує такі технології та інструменти:
-Nest.js
-TypeScript
-MongoDB

Інструкції з налаштування
Щоб запустити проект, виконайте наведені нижче дії.
1)Клонуйте репозиторій на локальну машину:
git clone <repository-url>
Замініть на фактичну URL-адресу сховища проекту.

2)Встановіть залежності проекту:
npm install

3)Запустіть сервер:
nodemon --exec ts-node src/main.ts
Після успішного запуску сервера ви побачите повідомлення про його статус.

4)Відкрийте веб-переглядач і перейдіть за такою URL-адресою:
http://localhost:<PORT>
Замініть <PORT>номер порту, указаний у .env файлі конфігурації.

5)Насолоджуйтесь додатком :)

Деталі проекту
Після доступу до хосту проекту ви перейдете на сторінку входу.

Проект включає дві ролі: адміністратора та менеджера .

За замовчуванням обліковий запис адміністратора доступний для автентифікації з такими обліковими даними:

-Електронна адреса : admin@gmail.com
-Пароль : admin
Після успішного входу ви будете перенаправлені на сторінку додатків. Відображені поля для кожної програми включають:
id, name, surname, email, phone, age, course, course_format, course_type, status, sum, alreadyPaid, created_at

Програми розбиті на сторінки, за замовчуванням 25 програм на сторінку, перераховані в порядку спадання.
Панель розмітки сторінок працює таким чином:
Коли ви на першій сторінці: Перша сторінка
Коли ви на останній сторінці: остання сторінка

Поточна сторінка має бути відображена в параметрах запиту. Під час доступу до проекту через параметр запиту вас має бути спрямовано на відповідну сторінку.

Упорядкування можна застосувати до кожного стовпця, клацнувши назву стовпця. Це запускає сортування в порядку зростання або спадання,
а вибір сортування зберігається в параметрах запиту.

Допомога Postman
Щоб зробити запит у Postman, виконайте такі дії:

Відкрийте Postman і виберіть відповідний метод запиту.
Введіть URL-адресу, замінивши :відповідний параметр фактичним значенням.
Налаштуйте всі необхідні заголовки або параметри для запиту.
Натисніть кнопку «Надіслати» (або натисніть гарячу клавішу «Enter»), щоб надіслати запит.
http://localhost:<PORT>Пам’ятайте, що для успішного підключення та отримання відповіді на вашому сервері має бути запущений сервер .
